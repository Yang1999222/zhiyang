# EE Lab Blog (Next.js App Router)

硬件/电子工程师风格个人博客模板，支持 Markdown 本地内容、标签、搜索、代码高亮与复制。

## 启动

```bash
npm install
npm run dev
```

访问 <http://localhost:3000>

## 内容目录

- 博客文章：`content/blog/*.md`
- frontmatter 字段：`title`, `date`, `tags`, `description`

## Vercel 部署

1. 推送项目到 GitHub。
2. 打开 Vercel，点击 **Add New... → Project**。
3. 导入仓库后保持默认配置（Next.js 自动识别）。
4. 点击 **Deploy**。
5. 首次部署后可绑定自定义域名。
